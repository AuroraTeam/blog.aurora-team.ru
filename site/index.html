MC Mirrors

Пользовательские зеркала для лаунчеров Minecraft

Добро пожаловать!

Хотели заняться созданием сборок для Minecraft, но не знали где их публиковать?
Наш сервис позволит Вам создать собственный файловый сервер с минимальными вложениями и без надобности в изучении различных технологий.

Список поддерживаемых нами зеркал

  • assets.mc-mirrors.ru
  • mirror.samopis.ru
  • exstra.mc-mirrors.ru

Хотите создать своё зеркало?

Для этого вам нужно отправить заявку на mr.catmanrus@gmail.com с просьбой поучаствовать в проекте.
Ограничение одно: можно хранить лишь файлы связанные с Minecraft (сборки, сервера, и др.), иначе вы будете исключены из проекта.
Отвечаю в течении суток, не забывайте переиодически проверять почту после отправки заявки.
